# Project Boot

このファイルは、新しいAIチャットでプロジェクトを復元するための起点ファイルである。

--------------------------------

## 復元手順

新しいAIチャットでは、必ず以下の順で参照すること。

1. AI_AGENT_SYSTEM.md
2. PROJECT_STATE.md
3. logic.md
4. rules.md（必要時のみ）

--------------------------------

## 読み取り目的

### 1. AI_AGENT_SYSTEM.md
以下を理解する
・AIの役割
・回答方針
・Cursorプロンプト形式
・Cursor報告形式
・禁止コマンド
・Allowlistコマンド
・変更禁止関数
・エンジン構造

### 2. PROJECT_STATE.md
以下を理解する
・現在のフェーズ
・現在の状態
・次にやること
・現在の実装前提

### 3. logic.md
以下を理解する
・複線図エンジンの構造
・各層の役割
・エンジン仕様

### 4. rules.md
必要時のみ参照する
・補足ルール
・追加保護ルール

--------------------------------

## 新チャットで使う復元指示

新しいAIチャットでは最初に以下を実行すること。

PROJECT_BOOT.md を読んでプロジェクトを復元してください。

--------------------------------

## 重要

・AIは PROJECT_BOOT.md の復元順序に従うこと
・長い引継ぎ文章より、ファイル参照を優先すること
・PROJECT_STATE.md を現在の真実として扱うこと
・コードベースと PROJECT_STATE.md がズレた場合は、コードベースを真実として PROJECT_STATE.md を更新すること
